# Codex Guide

Install the skill locally:

```bash
mkdir -p ~/.codex/skills
ln -s /path/to/artifactry/skills/artifactry ~/.codex/skills/artifactry
```

Use prompt:

```text
Use $artifactry to convert this Markdown into the requested export files. Read DESIGN.md if present, generate editable source artifacts, render final files, and validate before answering.
```

Good requests:

```text
Use $artifactry to turn bootcamp.md into a 16:9 PPTX and PNG slides using Terminal Operator style.
```

```text
Use $artifactry to turn worksheet.md into a polished DOCX and PDF.
```
