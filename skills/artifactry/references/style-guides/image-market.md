# Image Market

Image Market is a photo-led social and product storytelling style for consumer campaigns, social carousels, catalogs, launch posts, and visual narratives. It inherits from image-first discovery platforms, travel/commerce pages, music/media surfaces, and consumer retail systems.

## Core Idea

Let the image carry the feeling while the layout carries the argument. The artifact should be visually immediate, but captions, labels, and hierarchy must remain deterministic and readable.

## Visual Signature

- Large image zones with clean text overlays or caption bands.
- Friendly sans typography and simple editorial captions.
- Rounded image masks, masonry grids, or product tiles.
- Strong color accents only when tied to action or category.
- Alternation between full-bleed story frames and structured product/info frames.

## Palette

Palette depends on imagery. Use a neutral text system plus one brand-neutral accent derived from the image or content mood. Use overlays only to protect contrast. Do not tint every image with the same filter.

## Typography

Use a clean sans. Headlines should be short and social-readable. Captions should be compact but not tiny. Metadata can be small but must remain legible after mobile compression.

## Layout System

Use image-first compositions: hero image with title, two-image comparison, product grid, caption strip, quote over photo, and gallery closer. Maintain safe text zones. If no images are provided, create simple editorial image placeholders or use generated non-text assets.

## Slide Recipes

- Cover: full image, large title, small context label.
- Product/story card: image, title, 2-line caption.
- Gallery: 3 to 6 images with numbered captions.
- Comparison: two images or states with identical labels.
- Tip card: image top, actionable text bottom.
- Quote: portrait or atmospheric image plus quote block.
- Catalog: product tiles, tags, price/attribute if provided.
- Closer: final image plus call to action.

## DOCX Translation

DOCX should use image captions, product tables, section dividers, and clean page rhythm. Do not overuse full-bleed imagery in Word; export PDF if the user wants image-heavy design.

## Agent Rules

- Ask for images or generate non-text image assets if visual storytelling is central.
- Never let text sit on a busy image area without overlay protection.
- Keep each social frame self-contained.
- Use captions as part of design, not as afterthought.
- Validate image dimensions and crop safety.

## Avoid

Avoid stock-like dark blurred backgrounds, decorative photos unrelated to content, tiny overlay text, and generic product cards without visual specificity.
