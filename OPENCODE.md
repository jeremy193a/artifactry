# OpenCode Guide

Use this repository as a local skill/reference pack for Markdown export work.

Recommended instruction:

```text
Use the skill in skills/md-export-suite. Convert Markdown into polished output files, using DESIGN.md/getdesign.md style guidance when available, and run validation scripts before delivery.
```

For image/deck work, prefer:

```text
Markdown -> slide plan -> HTML/CSS fixed canvas -> PNG/JPG -> PPTX
```

For document work, prefer:

```text
Markdown -> normalized Markdown -> reference.docx -> DOCX/PDF
```
