# Artifactry

## Markdown In, Artifacts Out

Artifactry turns one Markdown source into polished files an AI agent can deliver.

- DOCX reports and handouts
- PPTX decks assembled from designed slide images
- PNG/JPG carousels for social publishing

---

## Agents Ask First

The skill teaches agents to clarify the output before rendering.

- What artifact do you need?
- Which style and aspect ratio should it use?
- Should it be a document, deck, carousel, or docs page?

---

## Preflight Before Export

Users should not manually guess dependencies.

- Run the requirement check
- Install Python packages when needed
- Ask before installing system tools

---

## Style Is A System

Artifactry exposes 10 generic style archetypes derived from a deep DESIGN.md corpus.

- Clear names instead of brand names
- Reusable tokens for documents and slides
- Compatible with getdesign.md inspiration files

---

## One Source, Many Formats

The agent keeps Markdown as the source of truth, then validates every exported file.

- Render images
- Build presentation files from images
- Validate DOCX, PPTX, PDF, PNG, and JPG
