# Markdown Is The New Source File

## Slide 1 — Hook

Your Markdown should not stop at notes. It can become a document, a deck, a PDF, or a social carousel.

## Slide 2 — Problem

Raw conversion creates generic files. Agents need a repeatable export workflow.

## Slide 3 — System

One Markdown source. Multiple output routes. One style system.

## Slide 4 — Style

DESIGN.md gives the agent mood, tone, typography, color, spacing, and component logic.

## Slide 5 — Result

The output is not just converted. It is shaped, rendered, validated, and ready to use.
